¿Qué es Microsoft Excel?
🟢 Principiante⏱ 8 min💰 +10 coins

📌 Definición

Microsoft Excel es una hoja de cálculo electrónica que permite organizar datos en una cuadrícula de filas y columnas, realizar cálculos automáticos y crear gráficos para analizar información de forma eficiente.

💡Excel puede manejar desde listas sencillas hasta modelos financieros complejos con millones de datos.

🏗 Estructura básica

Un archivo Excel se llama Libro de trabajo (.xlsx). Dentro hay Hojas de cálculo con una cuadrícula de:

  • Filas: numeradas del 1 al 1.048.576
  • Columnas: letras de A a XFD (16.384 columnas)
  • Celdas: intersección fila+columna → A1, B5, C12…

Selección y edición de celdas
🟢 Principiante⏱ 10 min💰 +12 coins

🖱 Formas de seleccionar

  • Una celda: Clic directo
  • Rango continuo: Clic y arrastrar, o Shift+clic en la última
  • No contiguas: Mantén Ctrl y haz clic en cada celda
  • Columna entera: Clic en la letra de cabecera
  • Fila entera: Clic en el número de la fila
  • Toda la hoja: Ctrl+E o clic en la esquina superior izquierda

✏️ Modos de edición

  • Escritura directa: Escribe sobre la celda, reemplaza el contenido
  • Editar existente: F2 o doble clic — sitúa el cursor dentro
  • Cancelar: Esc — descarta los cambios
  • Confirmar: Enter (baja), Tab (derecha), clic en ✓

🧹 Borrar datos

  • Supr: Borra el contenido, conserva el formato
  • Inicio → Borrar → Borrar todo: elimina contenido y formato
  • Inicio → Borrar → Borrar formatos: solo limpia el estilo visual
Copiar, cortar y mover
🟢 Principiante⏱ 8 min💰 +10 coins

📋 Copiar y pegar

  1. Selecciona → Ctrl+C (aparece línea punteada)
  2. Ve al destino → Ctrl+V

✂️ Cortar y pegar (mover)

  1. Selecciona → Ctrl+X
  2. Ve al destino → Ctrl+V
⚠️Al mover (Cortar+Pegar), las referencias relativas en fórmulas NO se ajustan. Al copiar, sí se ajustan automáticamente.

🎛 Pegado especial — Ctrl+Alt+V

  • Solo valores: Pega el resultado numérico, no la fórmula
  • Solo formatos: Aplica el estilo sin copiar datos
  • Transponer: Convierte filas en columnas y viceversa
  • Operación: Suma/resta/multiplica con los valores existentes

🤏 Autorrelleno

El controlador de relleno es el cuadradito en la esquina inferior derecha de la selección. Arrástralo para copiar o continuar series automáticamente (1,2,3… o Lunes,Martes…).

Validación de datos
🟡 Intermedio⏱ 12 min💰 +15 coins

🛡 ¿Qué es?

La validación de datos restringe qué valores se pueden introducir en una celda. Es esencial para formularios y para evitar errores de entrada.

⚙️ Cómo configurarla

  1. Selecciona la celda o rango
  2. Pestaña Datos → Validación de datos
  3. En "Permitir" elige: Número entero, Decimal, Lista, Fecha, Hora, Longitud o Personalizada
  4. Configura la condición (entre, mayor que, igual a…)
  5. Opcionalmente: configura Mensaje de entrada y Mensaje de error

📋 Lista desplegable (muy utilizado)

Validación → Permitir: Lista → Escribe las opciones separadas por ";" en Origen:

Sí;No;En proceso;Cancelado;Pendiente
💡Las listas desplegables son perfectas para hojas compartidas. Evitan inconsistencias y errores tipográficos.

Fórmulas en Excel
🟢 Principiante⏱ 12 min💰 +15 coins

⚡ ¿Qué es una fórmula?

Una fórmula es una instrucción de cálculo que empieza siempre con =. Excel la evalúa y muestra el resultado en la celda.

=A1+B1
=C3*0.21
=D5-D4
=SUMA(A1:A10)

📐 Reglas de sintaxis

  • Siempre empieza con =
  • Puede contener: referencias de celdas, números, texto entre comillas, operadores, funciones
  • Se recalcula automáticamente cuando cambia cualquier celda referenciada
💡Puedes ver y editar la fórmula de una celda en la barra de fórmulas, aunque la celda muestre el resultado calculado.

Funciones lógicas: SI, Y, O
🟡 Intermedio⏱ 15 min💰 +22 coins

🔀 Función SI

=SI(condición, valor_si_verdadero, valor_si_falso)

=SI(A1>=5, "Aprobado", "Suspenso")
=SI(B2>=18, "Mayor de edad", "Menor")
=SI(C3="", "Sin datos", C3)

🪆 SI anidado

=SI(A1>=9,"Sobresaliente",
SI(A1>=7,"Notable",
SI(A1>=5,"Aprobado","Suspenso")))
ℹ️En Excel 365 usa IFS en lugar de SI anidado: =IFS(A1>=9,"Sobresaliente", A1>=7,"Notable", A1>=5,"Aprobado", VERDADERO,"Suspenso")

🔗 Y y O combinados con SI

=SI(Y(A1>=18, B1="Sí"), "Puede votar", "No puede")
Y: TODAS las condiciones deben ser verdaderas

=SI(O(A1="VIP", B1>1000), "Descuento 20%", "Normal")
O: basta con que UNA condición sea verdadera

=SI.ERROR(BUSCARV(A1,B:C,2,0), "No encontrado")
SI.ERROR: captura cualquier error de la fórmula
BUSCARV, BUSCARH y XLOOKUP
🔴 Avanzado⏱ 20 min💰 +30 coins

🔍 BUSCARV

=BUSCARV(valor_buscado, tabla, número_columna, FALSO)

=BUSCARV(A2, $E$2:$G$100, 2, FALSO)
← Busca A2 en la 1ª columna de E2:G100
← Devuelve el valor de la columna 2
← FALSO = coincidencia exacta (casi siempre)
⚠️BUSCARV solo busca hacia la derecha. Para buscar en cualquier dirección usa ÍNDICE+COINCIDIR o XLOOKUP.

🆕 XLOOKUP (Excel 365/2021)

=XLOOKUP(valor, rango_búsqueda, rango_resultado, [si_no_encontrado])

=XLOOKUP(A2, E:E, G:G, "No encontrado")
← Más flexible: busca en cualquier dirección
← Acepta valor por defecto si no encuentra

🔧 ÍNDICE + COINCIDIR (clásico)

=ÍNDICE(G:G, COINCIDIR(A2, E:E, 0))
← COINCIDIR: posición de A2 en columna E
← ÍNDICE: valor en esa posición de G
← Funciona en CUALQUIER dirección

🗂 Tablas de Excel y gestión de datos
🟡 Intermedio⏱ 20 min💰 +25 coins

📋 ¿Qué es una Tabla de Excel?

Insertar → Tabla (o Ctrl+T) convierte un rango en un objeto estructurado con ventajas únicas:

  • Filtros automáticos en los encabezados
  • Estilos visuales con bandas de color alternadas
  • Fila de totales con funciones predefinidas
  • Referencias por nombre: =Tabla1[Ventas] en vez de =$B$2:$B$100
  • Se expanden automáticamente al añadir datos
  • Las fórmulas se copian solas a toda la columna

🔍 Filtrar y ordenar datos

Usa las flechas de los encabezados para filtrar por valor, color, texto o número. Para criterios más complejos: Datos → Filtro avanzado.

🧹 Eliminar duplicados

Datos → Quitar duplicados. Elige las columnas a comprobar. Imprescindible para limpiar bases de datos importadas.

🌀 ¿Qué son?

Las tablas dinámicas resumen, agrupan y analizan grandes volúmenes de datos de forma interactiva, sin escribir una sola fórmula. Son la herramienta más potente de Excel para análisis de datos.

⚡ Crear una tabla dinámica

  1. Clic en cualquier celda de los datos
  2. Insertar → Tabla dinámica
  3. Elige si insertarla en hoja nueva o en la actual
  4. En el panel de campos, arrastra a: Filas, Columnas, Valores, Filtros

🎛 El panel de campos

  • Filas: Agrupa verticalmente (por mes, región, categoría…)
  • Columnas: Agrupa horizontalmente
  • Valores: Qué calcular (suma, promedio, máximo, conteo…)
  • Filtros: Filtro global para toda la tabla dinámica

🔪 Segmentadores (Slicers)

Filtros visuales con botones. Insertar → Segmentación de datos. Se pueden conectar a múltiples tablas dinámicas simultáneamente para crear dashboards interactivos.

💡Las tablas dinámicas NO modifican los datos originales. Son una vista dinámica. Puedes tener varias sobre el mismo origen de datos.

⚙️ Macros y VBA

  • Grabar macro: Vista → Macros → Grabar macro. Excel registra tus acciones como código VBA.
  • Editor VBA: Alt+F11. Edita, mejora o crea macros desde cero.
  • Guarda el archivo como .xlsm para conservar las macros.
Sub FormatearTabla()
Range("A1:E1").Interior.Color = RGB(33,115,70)
Range("A1:E1").Font.Color = RGB(255,255,255)
Range("A1:E1").Font.Bold = True
MsgBox "¡Tabla formateada!"
End Sub

🔌 Power Query

Datos → Obtener y transformar datos. ETL (Extracción, Transformación y Carga) sin programar:

  • Importa desde Excel, CSV, web, SQL Server, APIs REST…
  • Transforma: divide columnas, filtra filas, pivota/anula el pivote, une tablas
  • El proceso se puede refrescar automáticamente con nuevos datos

🆕 Matrices dinámicas (Excel 365)

FunciónDescripciónEjemplo
FILTERFiltra y devuelve un array dinámico=FILTER(A:C,B:B>100)
SORT / SORTBYOrdena dinámicamente=SORT(A1:C10,2,-1)
UNIQUEDevuelve valores únicos=UNIQUE(A:A)
LETDefine variables dentro de una fórmula=LET(x,A1*2,x+B1)
LAMBDACrea funciones personalizadas sin VBA=LAMBDA(x,x^2)

📊 Power Pivot

Para modelos de datos con millones de filas. Permite crear relaciones entre tablas como en una base de datos relacional. Usa el lenguaje DAX para medidas calculadas avanzadas.
